Construindo um app para treinos de boxe

Este app era um exercício de JavaScript com intenção de fazer um cronometro. Sempre uso apps que contão rounds de boxe para fazer alguns treinos e sempre acho os apps feitos propositadamente feitos para dificultar o uso. Quando não a dificuldades para usar o app a interface é bem tosca.

Com o exercício em mãos resolvi ir um pouco além e customizar uma interface clean e apliar a funcação do cronometro para um contador de raunds. Inicialmente foi um tanto facil, pois usando HTML e Tailwind rapidamente criamos uma interface. Fazer a ligação entre o cronometro e o JavaScript foi trivial, contudo, alguns bugs aconteceram no processo de customizar o contador.

Após algum esforço e paciência consegui concluir o app e obviamente agora estou o usando o meu!
